
# Covid 19 App

web-based application that displays data on covid in the world and every province in Indonesia


## Run Locally

Clone the project

```bash
  git clone https://github.com/NdkVin/covid-19.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start-dev
```
